# Claude rules — ArgusGraph

## Commits

- **Never add AI attribution to commits.** No `Co-Authored-By: Claude ...`, no
  "Generated with Claude Code" — no self-quoting of any kind in commit messages,
  PR bodies, or code comments.

## Documentation

- **Never commit design specs or the `docs/` directory.** Specs and research notes
  are local working artifacts (`docs/` is gitignored). Durable decisions go into
  `.planning/` instead.
